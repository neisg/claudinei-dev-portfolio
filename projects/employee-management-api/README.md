# Employee Management API

REST API developed using Java and Spring Boot to manage employees.

## 🚀 Features
- Create, update, delete and list employees
- DTO pattern implementation
- Exception handling
- RESTful API architecture
- MySQL database integration
- Docker support

## 🛠 Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Docker
- Maven

## 📌 Purpose
This project demonstrates backend development skills, clean architecture and REST API best practices.
